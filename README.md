# 乒乓球赛制模拟器

## 这是什么？

这是一个简单的乒乓球比赛模拟器，用来比较11分制和21分制下，中国队获胜的概率。

## 功能介绍

- **模拟11分制**：先到11分，领先2分获胜
- **模拟21分制**：先到21分，领先2分获胜
- **比较两种赛制**：同时模拟两种规则，看哪种对中国队更有利
- **可视化结果**：网页版有进度条显示胜率

## 简单原理

程序会模拟很多场比赛（比如10万场），每场比赛随机决定每一分谁赢（根据你设置的概率）。然后统计中国队赢了多少场，算出胜率。

## 注意事项

- 模拟次数越多，结果越准确，但电脑跑得慢一点
- 概率设置在0.5到0.95之间比较合理
- 这只是模拟，不是真的比赛结果
